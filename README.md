# get-tron-address

- ```bash
  # 随机生成地址和私钥: 
  get-tron-address-mac seed ''
  # 根据助记词获取地址和私钥(空格或逗号隔开)：
  get-tron-address-mac
  或
  get-tron-address-mac seed 'addict,mask,afraid,cushion,river,guard,beauty,quarter,judge,that,nerve,father'
  # 根据私钥获取地址:
  get-tron-address-mac key 85e40fae8c922fbd34858707f685fa626a43af6d2b4e736f08b98bc907e065a5
  ```
